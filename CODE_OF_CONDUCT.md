# Code of Conduct

Coral adopts the [Contributor Covenant, version 2.1][cov] as its Code of
Conduct. The canonical text is maintained by the Contributor Covenant
project and is incorporated here by reference.

[cov]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Scope

This Code of Conduct applies in all project spaces — issues, pull requests,
discussions, and any other communication channel run by the project — and
when an individual is officially representing the project in public spaces.

## Reporting

If you observe a violation of this Code of Conduct, please report it via
**GitHub's private vulnerability reporting** for this repository:

  https://github.com/noelschwarz/coral/security/advisories/new

The advisory channel is private and the maintainers will see it. Using it for
Code of Conduct reports (in addition to security vulnerabilities) keeps the
reporting surface small and confidential. Mark the report subject clearly
with "Code of Conduct" so we can route it correctly.

All reports will be reviewed and investigated promptly and fairly. The
maintainers are obligated to respect the privacy and security of the
reporter of any incident.

## Enforcement

Maintainers will follow the [Contributor Covenant 2.1 enforcement
guidelines][enforce] when determining the consequences of any action
they deem in violation of this Code of Conduct.

[enforce]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines
